1   The whole project was written in Python 3.6

2   The bot workflow is explained in the second paragraph of the report. Note that you have to run only the main.py file.

3   The Classifiers are pretrained. If you want to test them, download the glove dictionary (5,6 Gb) at this link: 
    http://nlp.stanford.edu/data/glove.840B.300d.zip

4   The spacy model used is the en_core_web_md (1 GB) set as 'en_default' through the command:
    'python -m spacy link en_core_web_md en_default'
